# Shopee-api

Olá! Bem vindo a nossa api. Antes de tudo, olhe o conteúdo da pasta info para saber mais sobre as características base do sistema e saber as ferramentas necessárias para rodá-lo.

Se já estiver pronto para rodar o sistema, crie na raiz do projeto um arquivo ".env" e siga o .env.template para preencher ele corretamente.

Agora, ative o docker e suba os conteineres (docker compose up -d).

Por fim, ponha o spring para rodar o projeto (sugiro instalar a extensão Spring Boot Dashboard).
