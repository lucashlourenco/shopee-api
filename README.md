# Shopee-api

Olá! Bem vindo a nossa api. Aintes de tudo, olhe o conteúdo da pasta info para saber mais sobre as características base do sistema e saber as ferrsmentas necessárias para rodá-lo.

Se já estiver pronto para rodar o sistema, crie na raiz do projeto um arquivo ".env" esiga o .env.template para preencher ele coretamente.

Acora, ative o doker e suba os conteineres (docker compose up -d).

Por fim, ponha o sping para rodar o projeto (sugiro instalar a extensão Spring Boot Dashboard).
