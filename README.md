# Shopee-api

Olá! Bem vindo a nossa api. Antes de tudo, olhe o conteúdo da pasta info para saber mais sobre as características base do sistema e saber as ferramentas necessárias para rodá-lo.

Se já estiver pronto para rodar o sistema, crie na raiz do projeto:

 + Um arquivo ".env" esiga o .env.template para preencher ele corretamente.
 + Uma pasta, se não houver, camada: shopee-storage.

Agora, ative o doker e suba os contêineres (docker compose up -d).

Por fim, ponha o sping para rodar o projeto (sugiro instalar a extensão Spring Boot Dashboard).
